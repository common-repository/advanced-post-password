Enhance the security of password-protected posts/pages with this plugin.

== Description ==

This plugin offers advanced security features for password-protected posts/pages.

Key Features:
- Access multiple posts/pages without entering passwords repeatedly.
- Admins can bypass password entry for protected pages.
- Set a master URL to access all password-protected posts.
- Option to remove WordPress-added prefixes from private and password-protected pages.

To use, simply install and activate the plugin.

== Installation ==

Via WordPress Admin:

1. Navigate to Plugins -> Add New.
2. Search for "Advanced Post Password."
3. Install and activate the plugin.

Via FTP:

1. Download the plugin.
2. Unzip the .zip file and upload the folder to `/wp-content/plugins/`.
3. Activate the plugin via the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How do I use this plugin? =

After activation, the plugin functions automatically. Users won't need to input passwords for each unlocked page. It also removes "Protected" and "Private" from all page titles on your site.

By using the master URL, users can access pages without requiring additional passwords.
